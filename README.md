# Machine-Learning-Modeling-for-Menstrual-Cycle-Prediction-Using-mcPHASES-Data
Machine Learning Modeling for Menstrual Cycle Prediction Using mcPHASES Data and Conceptual Replication of German Study.


This paper focuses on predicting menstrual cycle phases using data from the mcPHASES dataset, which contains physiological and behavioral signals such as heart rate, sleep, and activity levels.


We took the following steps :
Data integration from multiple sources (wearables, logs) and combined physiological + behavioral + temporal data, 
preprocessing & feature engineering to extract meaningful signals, 
model training using machine learning algorithms, and 
evaluation of how well models predict different menstrual phases that can be applied to real-world settings.
